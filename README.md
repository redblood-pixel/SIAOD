## Инструкция по запуску

После копирования репозитория, необходимо перейти в директорию с проектом

```
cd kurs
```

Установить зависимости

```
python3 -m pip install -r requirements.txt
```

Запуск TUI с реализацией влоб

```
python3 tui_brute.py
```

Запуск TUI с генетическим алгоритмом

```
python3 tui_genetic.py
```
